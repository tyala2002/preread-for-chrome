🔍 Web記事検索API

要約・レビュー記事を検索するためのAPIを選択してください。
Tavily Search API が最も簡単に設定できるためおすすめです。

取得方法: Tavily → アカウント作成（無料・クレカ不要）→ APIキーをコピー

Programmable Search Engine で「ウェブ全体を検索」に設定したエンジンのIDを入力してください。

SerpAPI ダッシュボード でAPIキーを確認してください。

🎬 YouTube Data API （任意）

YouTube解説動画を検索するためのAPIキーです。
未設定の場合はYouTube動画の検索がスキップされますが、Web記事検索は正常に動作します。

取得方法: Google Cloud Console → APIとサービス → 認証情報 → APIキーを作成 → YouTube Data API v3 を有効化
