プライバシーポリシー — Preread

最終更新日：2026年2月23日

📋 概要

Preread は、ユーザーの個人情報を収集・保存・販売しません。 外部に送信するデータは「書籍タイトル」のみです。

Preread（以下「本拡張機能」）は、Amazonの書籍ページに関連する 要約・レビュー記事とYouTube動画を自動で収集し、 NotebookLM（Google LLC）のソースとして追加するChrome拡張機能です。

本ポリシーは、本拡張機能がどのようにデータを取り扱うかを説明します。

📤 外部サービスへ送信するデータ

本拡張機能は、ユーザーが操作を実行したときのみ、 以下のデータを各サービスへ送信します。 送信先は各サービスのサーバーのみであり、開発者のサーバーには送信しません。

送信先 送信されるデータ タイミング
Tavily Search API 書籍タイトル（検索クエリ） 「ソースを検索」ボタンを押したとき
YouTube（youtube.com） 書籍タイトル（検索クエリ） 「ソースを検索」ボタンを押したとき
NotebookLM（notebooklm.google.com） 選択したURL一覧・書籍タイトル・一時的な認証トークン 「NotebookLMに追加」ボタンを押したとき

※ NotebookLMへの認証トークンは、同サービスのページから一時的に取得し、 その操作のみに使用します。トークンは保存・外部送信しません。

💾 デバイス内に保存されるデータ

以下のデータはユーザーのブラウザ内にのみ保存されます。 開発者サーバーへの送信は行いません。

データ 保存場所 用途
Tavily APIキー chrome.storage.sync Web記事検索に使用
追加済み書籍の履歴
（書籍タイトル・ASINコード・NotebookLM URL・日時）		 chrome.storage.local ポップアップと設定画面での履歴表示に使用

※ chrome.storage.sync に保存されたデータは、 Googleアカウントでログイン中の場合、Chromeの同期機能によって ユーザー自身の他のデバイスと同期される場合があります。

🚫 収集しないデータ

  • 氏名・メールアドレスなどの個人識別情報
  • 閲覧履歴・検索履歴
  • 位置情報
  • 決済・金融情報
  • ユーザーの行動ログや分析データ

本拡張機能は広告ネットワーク、アナリティクスツール、 その他のトラッキングサービスを一切使用しません。 収集したデータを第三者に販売・共有することもありません。

🔑 Chrome 権限の使用理由

権限 使用理由
activeTab 現在開いているAmazonページのURLを取得し、書籍ページかどうかを判定するため
scripting AmazonページのDOMから書籍タイトルを取得するため
storage APIキーと書籍履歴をブラウザ内に保存するため
tabs NotebookLMのタブが既に開いているか確認するため
clipboardWrite 選択したURLをクリップボードにコピーするため
notifications NotebookLMへの追加完了をシステム通知でお知らせするため

📝 ポリシーの変更

本ポリシーの内容を変更する場合は、このページの「最終更新日」を更新します。 重要な変更が生じた場合は、拡張機能の更新履歴（Chrome ウェブストア）にも記載します。

変更後も本拡張機能を引き続きご利用いただいた場合、 変更後のポリシーに同意したものとみなします。
