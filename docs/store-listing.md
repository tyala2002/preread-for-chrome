# Chrome Web Store 掲載文

---

## 拡張機能名

```
Preread — Amazon書籍 × NotebookLM 自動ソース追加
```

---

## 短い説明（132文字以内）

```
Amazonの書籍ページを開くだけで、関連する要約・レビュー記事・YouTube解説動画を自動収集し、NotebookLMにワンクリックで追加します。
```

---

## 詳細説明

```
📚 本を読む前に、全体像を掴もう。

Preread は、Amazonで気になる本を見つけたとき、その本に関する情報を自動でリサーチしてNotebookLMにまとめてくれるChrome拡張機能です。


【こんな方におすすめ】

・読書前に要約・レビューを確認してから購入を決めたい方
・YouTube解説動画も含めて事前に全体像を掴みたい方
・NotebookLMを活用した読書・学習ワークフローを持っている方


【機能】

🔍 自動検索
Amazonの書籍ページを開いてボタンを押すだけで、以下を自動収集します：
  - 要約・レビュー・まとめ記事（最大5件）
  - YouTube解説・要約動画（最大3件）

✅ 選んでまとめて追加
収集した候補をチェックボックスで選択し、「NotebookLMに追加」ボタン1つで一括追加。
追加中はリアルタイムで進捗が表示されます。

📋 クリップボードコピー
自動追加が使えない場合でも、選択したURLをコピーしてNotebookLMに手動で追加できます。

✏️ タイトル編集
自動取得したタイトルを手動で修正することも可能です。


【使い方】

1. 拡張機能の設定画面でAPIキーを設定
   - Tavily API（無料・クレカ不要）：Web記事検索に使用
   - YouTube検索はAPIキー不要で動作します

2. Amazonの書籍ページを開く

3. 拡張機能のアイコンをクリック

4. 「ソースを検索」ボタンを押す

5. 追加したいソースを選んで「NotebookLMに追加」


【対応サービス】

・Amazon.co.jp / Amazon.com
・NotebookLM（notebooklm.google.com）


【注意事項】

・NotebookLMの仕様変更により、動作しなくなる場合があります。
・NotebookLMへのログインが必要です。
・APIキーはお使いのブラウザにのみ保存され、外部サーバーには送信されません。
```
